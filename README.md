# Rust sandbox

## Getting started

```sh
# build project
$ cargo build
# run binary
$ cargo run
# check project (faster than build but does not generate build)
$ cargo check
# run test
$ cargo test
# watch
$ cargo install cargo-watch
$ cargo watch -x check -x test -x run
```
